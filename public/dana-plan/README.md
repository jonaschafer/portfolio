# Dana plan viewer (static)

Served at **`/dana-plan/`** on the portfolio site. Open `https://<your-vercel-url>/dana-plan/` (or `/dana-plan/view-dashboard.html` for at-a-glance) on your phone or computer.

**To update the plan:** Regenerate `plan-data.json` in the Dana plan repo (`Dana plan/viewer/v1`: run `node parse-plan.js`), then copy `plan-data.json` into this folder and redeploy the portfolio.
