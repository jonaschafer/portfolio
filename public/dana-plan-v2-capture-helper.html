Calendar screenshot → clipboard

Use this so you can paste the calendar into Figma.

1. With dev server running, open the calendar: /dana-plan-v2/calendar
2. Either take a screenshot yourself (Cmd+Shift+4) or use Chrome DevTools MCP: navigate to /dana-plan-v2/calendar, capture (screenshot saved to public/cal-capture.png), then click below to copy that image to clipboard.
3. In Figma, open your file and the frame (e.g. node 62-469). Press Cmd+V (Mac) or Ctrl+V (Win) to paste the image.

MCP flow: 1) Navigate to calendar, 2) take_screenshot with filePath public/cal-capture.png, 3) Navigate to this helper, 4) evaluate_script to click #copyBtn (or run the fetch+clipboard script), 5) Navigate to Figma, 6) evaluate_script to dispatch Cmd+V / paste.
