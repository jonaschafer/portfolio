## Design Matches Figma Blog

**Colors:**
- Figma Blue (#0D99FF) for links and accents
- Dark text (#18191B) for body
- Light gray (#777777) for meta/secondary text
- Soft border (#E5E5E5) for dividers

**Typography:**
- **Font:** Inter (same as Figma)
- **Body:** 18px, line-height 1.7
- **Headings:** Large (3xl-6xl), tight letter-spacing (-0.02em)
- **Max width:** 720px for comfortable reading

**Layout:**
- Clean, spacious with generous whitespace
- Minimal navigation
- Large titles with strong hierarchy
- Tags as clickable blue text
- Center-aligned blockquotes (like Figma's pull quotes)

**Special Features:**
- Mermaid diagrams styled with Figma blue
- Tables with clean borders
- Code blocks with light gray background
- Blue selection highlight
